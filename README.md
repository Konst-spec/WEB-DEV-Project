A project in the discipline of web development.
This project is a web application designed for students to leave reviews about university professors and evaluate them based on different criteria such as rating, difficulty, and personal feedback.
The main goal of the platform is to provide a convenient and transparent environment where students can:
browse academic subjects
explore professors teaching those subjects
read reviews from other students
share their own learning experience

Subjects:
Retrieve a list of all subjects
View details of a specific subject
Get a list of professors associated with a subject

Professors:
View professors filtered by subject
Access basic information about each professor
Mark professor as preferable to add them to wishlist (wishlist is a page that contains all the profesors that preferable for specific subject. You can add multiple professors to one subject and choose who is more preferable)

Reviews:
You can add whole description, ratiing out of 5, dificulty, own experience of professor.
You can add anonimous option.
User can leave only one review per professor per subject.

Pages:
Page with list of all subjects, from which you can go to the page of specific subject.
Page with specific subject, where you can see the list of all professors, that teach this subject.
Page with list of all professors.
Page of specific professor, that will show information about them and list of all reviews(You can filter reviews on that page by positive, negative)
Wishlist page with your preferrable professors.
